Eggnog Software

Eggnog can be produced in 4.6 and up, while Snags & Glaces can be produced in 4.

It is possible to produce Eggnog in two ways.

1. Use Eggnog for a 1.4Gb file as opposed to 3GfG and the 1.7GfEgg-2.0.x file as it is possible to convert them from their standard output, however this will require having a 2.2g file, meaning there will be around 400,000 eggs left to supply the files required to run the 3GfEggnog on 3.3g of disk space.

2. Use the Eggnog.exe to get the source files needed for the 3GfEggnog.

It is also possible to use multiple ePATA files instead of one, for example to produce 3.9Gb or 4Gb files.

This section does not seek to cover those issues I have encountered in using the 3GbEggnog method. However, the source file used in the procedure above will be the same as the 3.9GfEgg NGP file used to produce the 3.9Gb and 4Gb files. Thus making every EPGM file with this file set to this file type, even if it contains multiple ePATA files, would give the same results.
